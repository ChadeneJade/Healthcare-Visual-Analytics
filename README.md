# Healthcare Visual Analytics Dashboard

Interactive data visualization dashboard for healthcare analytics using D3.js and ArcGIS API.

## Project Overview

This project is a comprehensive healthcare visual analytics tool developed as part of the Information Visualization course at ESI.

**Students:** Belkadi Chadene, Ghecham Sarah  
**Supervisor:** Prof. Samiha Fadloun  
**Academic Year:** 2025-2026

## Features

- **4 Interactive D3.js Visualizations:**
  - Donut chart for test results distribution
  - Heatmap for medical conditions vs test results
  - Area chart for billing amount analysis
  - Grouped bar chart for patient demographics

- **Interactive Map:** ArcGIS-powered geospatial visualization of hospitals
- **Dynamic Filtering:** Filter by gender and age group
- **Responsive Design:** Works on desktop and mobile devices

## How to Run

1. Clone the repository:
```bash
   git clone https://github.com/votre-username/healthcare-visual-analytics.git
```

2. Open `index.html` in a modern web browser



## Technologies Used

- HTML5, CSS3, JavaScript (ES6+)
- D3.js v7 - Data visualization
- ArcGIS JavaScript API v4.26 - Geospatial mapping
- Healthcare Dataset from Kaggle
